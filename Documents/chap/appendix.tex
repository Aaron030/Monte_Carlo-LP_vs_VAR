\subsection{Code and Data Availability}

All code and replication materials for both extensions are available at
\href{https://github.com/Aaron030/MonteCarloSeminar}{github.com/Aaron030/MonteCarloSeminar}.
The repository contains the simulation code, the routines that generate every figure, table summaries of the figures,
and instructions to reproduce the full study.
